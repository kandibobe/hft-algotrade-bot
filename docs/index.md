# Welcome to Stoic Citadel

This is the central documentation for the Stoic Citadel MFT Algorithmic Trading Bot.

## Overview

Stoic Citadel is a hybrid Mid-Frequency Trading (MFT) system designed for algorithmic trading on cryptocurrency exchanges. It leverages a dual-layer architecture to combine strategic, synchronous analysis with high-speed, asynchronous execution.
