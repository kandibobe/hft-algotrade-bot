# Security Policy

## Supported Versions

We are committed to ensuring the security of Stoic Citadel. Security updates are applied to the most recent major release.

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

The Stoic Citadel team and community take all security vulnerabilities seriously. Thank you for improving the security of our project. We appreciate your efforts and responsible disclosure and will make every effort to acknowledge your contributions.

To report a security vulnerability, please email the project lead at `security@stoic-citadel.com` (this is a placeholder, please replace with a real contact).

You should receive a response within **48 hours**. If for some reason you do not, please follow up via the same email to ensure we received your original message.

Please include the following information in your report:

-   A clear and concise description of the vulnerability.
-   The component or location of the vulnerability.
-   Steps to reproduce the vulnerability.
-   Proof-of-concept or exploit code, if available.
-   Any potential impact of the vulnerability.
-   Your name and a link for recognition (e.g., Twitter or GitHub profile).

**Please do not report security vulnerabilities through public GitHub issues.**

## Disclosure Policy

When the team receives a security bug report, they will assign it to a primary handler. This person will coordinate the fix and release process, involving the following steps:

1.  **Confirm the vulnerability:** The team will verify the reported vulnerability and determine its potential impact.
2.  **Code audit:** An audit will be performed to find any similar issues.
3.  **Prepare a fix:** A fix will be developed and tested.
4.  **Release:** The fix will be released as part of the next available patch or minor version. The security advisory will be published on GitHub.

We will endeavor to keep you informed about our progress throughout the process.
